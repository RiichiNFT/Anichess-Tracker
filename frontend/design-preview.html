Changes in this preview
1 · Type Scale
Collapsed ~30 one-off rem values into 7 named CSS vars (--ts-xs through --ts-2xl). Every text element now maps to a step.
2 · Visual Hierarchy
Event name: 1.75rem (was 3.1rem). "Leaderboard" card title: 1.2rem (was 4rem). Clear anchor, no two elements competing for dominance.
3 · Badge System
9 bespoke variants → chip (6px) for status, pill (20px) for category. 3 semantic colors: blue=interactive, green=qualified, gold=achievement.
4 · Nav Clarity
Internal links: 600 weight, white. External links: 400 weight, dimmed, ↗ at full opacity. Font size reduced from 1.14rem to 0.82rem.
5 · Eligibility Banner
Cramped horizontal strip → 3-col grid. Rule / CTA / Deadline each gets its own column. Tooltip uses <details> (touch-friendly). Note is italic, not all-caps.
6 · "See Below" Removed
Replaced the meaningless 1.5rem "See below" text in the qualified section with a real anchor link to the leaderboard table.
7 · Qualify Badge Color
"Top 8 Qualifies" badge changed from accent blue to green — blue is reserved for interactive elements, green signals achievement/status.
8 · Footer
Added branding, useful outbound links (Discord, Lichess, Anichess.com), and moved "refreshes every 30 min" into context near "Data from API".
Not shown in this preview: the expand-button label fix (mobile), inline-style cleanup, and past-events.html page-specific styles moving into style.css — those are code-level changes with no visual difference to preview.
