Anichess Rising Stars Tournament #2

UX Designer Opinion
Works Well
Page-level hero titles
The 300/800 contrast technique looks strong on single-purpose headings — "Past Events", "Tournament #2", the replay page. It gives the site a distinct editorial identity that standard weight-700 headings don't have. The wide tracking adds premium feel.
Use With Care
Event name on home page
Works if the line break is intentional (descriptor light / identity bold). The risk is that "Anichess Rising Stars" in 300 weight reads as secondary, when it's actually the full name. The split needs to feel natural, not arbitrary.
Not Recommended
Card headers + nav links
"Leaderboard" with a gradient in a card header is over-designed for a functional label. Nav links at 300 weight + 0.10em tracking feel spaced out and slow to scan. The technique works as an accent, not a system-wide rule.
Recommendation: Selective, not global.
  • Apply it to page hero titles only — "Past Events", "Tournament Details", and the main event name. These are the moments that earn the editorial treatment.
  • Keep card headers (Leaderboard, About This Tournament) at weight 600–700, no gradient — they are functional labels, not brand moments. Applying gradient text to a card header competes with table data below it.
  • Keep nav at weight 600, normal tracking — uppercase + wide tracking in a nav bar makes each link harder to scan quickly.
  • Logo could adopt it — thin "ANICHESS" / bold gradient "TRACKER" is a clean application and adds polish to the header without disrupting the page body.
